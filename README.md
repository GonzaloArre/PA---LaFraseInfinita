# PA---LaFraseInfinita

LaFraseInfinita es un sistema de gestión para una biblioteca con un enfoque a un entorno local, que nos permite el registro, modificación y baja de socios. Ademas nos permite registrar nuevos ingresos de libros y tanto el registro como la devolución de prestamos.
